# Blue Tape Sites Deployment

Blue Tape Sites is now set up to deploy from this GitHub repository to Vercel.

- Production project: `blue-tape-sites`
- Production preview domain: `https://blue-tape-sites.vercel.app`
- Build command: `pnpm build`
- SEO assets are generated during the build by `scripts/generate-seo-assets.ts`.
- Static SEO route snapshots are generated during the build by `scripts/prerender-static-routes.ts`.
