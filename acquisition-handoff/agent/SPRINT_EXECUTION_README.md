**Author:** Manus AI

## Objective

The objective of the next sprint is to move Blue Tape Sites from strategy into active market execution. The sprint should produce three concrete outputs: two published niche landing pages, one working prospect list of fifty qualified contractor companies, and one live partner outreach system that can begin generating warm introductions.

This sprint is successful if Blue Tape Sites finishes the period with real distribution, not just more planning. That means the next agent should prioritize assets that can be used immediately by outbound outreach and partner conversations.

## Non-Negotiable Constraints

Blue Tape Sites should stay narrow during this sprint. The next agent should not add extra niches, extra offers, or extra channels unless a clear reason emerges from live signal. The first sprint is meant to answer a small number of practical questions well.

| Constraint | Rule |
|---|---|
| **Niches** | Focus only on plumbers and remodelers for this sprint |
| **Geography** | Start with Southern California and nearby obvious markets only |
| **Channels** | Prioritize niche pages, outbound micro-audits, and partner outreach |
| **Tone** | Keep all copy direct, premium, contractor-aware, and low on jargon |
| **Offer** | Lead with clarity, trust, and conversion improvement rather than generic “marketing” language |

## Decision Rules During Execution

The next agent should use simple rules instead of improvising constantly.

| Situation | Decision Rule |
|---|---|
| **One niche gets clearer traction** | Concentrate harder on that niche instead of broadening |
| **Prospects reply but object to price** | Review fit and proof before changing pricing language |
| **Prospects ignore the message entirely** | Change the problem framing, not the whole strategy |
| **Partners respond better than cold prospects** | Increase effort on partnerships while keeping a smaller outbound test running |
| **Landing pages feel too generic** | Rewrite to sound more trade-specific before scaling outreach |

## Messaging Priorities

The next agent should keep message framing tightly connected to contractor pain points.

For **plumbers**, the strongest likely themes are mobile trust, urgency, clarity, and booked calls. The tone should imply that a weak site loses confidence exactly when a homeowner is ready to choose someone fast.

For **remodelers**, the strongest likely themes are premium presentation, project credibility, better qualification, and reducing wasted consultations. The tone should imply that the website should pre-sell craftsmanship before the owner ever gets on a call.

## What Not to Do

The next agent should avoid broad, unfocused activity that feels productive but does not create signal. That means no giant lead list scraped from everywhere, no generic outreach template copied across all trades, and no sudden expansion into paid ads unless there is already strong evidence that the funnel converts.

## Close-Out Requirement

At the end of the sprint, the next agent should leave behind a clear review of what worked, what did not, which niche responded better, which message angle produced the strongest response, and what the second sprint should do differently. The output should make the next decision easier, not restart planning from zero.
