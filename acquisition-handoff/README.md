## Purpose

This package is the operational handoff for the next Blue Tape Sites acquisition sprint. It translates the earlier customer-acquisition strategy into files that a future agent can execute in order without having to re-derive the plan from scratch.

The recommended operating model remains the same: **use strategic partnerships and precision outbound to generate the first qualified conversations, while building a small number of supporting niche inbound assets in parallel**. The goal of this package is not theory. It is to help the next run ship the first practical GTM assets quickly and consistently.

## Recommended Execution Order

The next agent should execute this work in a deliberate sequence rather than trying to do everything at once. The first move is to read the sprint execution brief, because that file defines the order of work, success criteria, and decision rules. The second move is to create the niche landing pages, since those pages give both outbound prospects and partners somewhere relevant to land. The third move is to use the prospecting files to build the first target list and attach short teardown notes. The fourth move is to begin partner outreach using the templates and one-pager.

| Step | What to Do | Primary Files |
|---|---|---|
| **1** | Read the overall sprint brief and confirm sequencing | `agent/SPRINT_EXECUTION_README.md` |
| **2** | Build the first two niche landing pages | `landing-pages/PLUMBER_LANDING_PAGE_BRIEF.md`, `landing-pages/REMODELER_LANDING_PAGE_BRIEF.md` |
| **3** | Assemble and score the first 50 prospects | `prospecting/FIRST_50_PROSPECT_TEMPLATE.csv`, `prospecting/WEBSITE_TEARDOWN_WORKFLOW.md` |
| **4** | Start partner outreach and referral setup | `partnerships/REFERRAL_ONE_PAGER.md`, `partnerships/PARTNER_OUTREACH_TEMPLATES.md` |
| **5** | Review signal, adjust niche/message angle, and continue the sprint | All files together |

## How to Use This Package Well

The most important instruction is to stay narrow. Blue Tape Sites should not try to target every contractor at once. The first sprint should focus on two niches only, use a small number of clear message angles, and treat every outreach reply as information about positioning. If the next agent gets strong signal from one niche and weak signal from the other, the right move is to concentrate harder, not broaden the campaign.

The next agent should also preserve the Blue Tape Sites brand voice throughout. Every page, email, and referral asset should feel direct, premium, and contractor-aware. The copy should speak to trust, clarity, credibility, and booked calls rather than abstract marketing jargon.
